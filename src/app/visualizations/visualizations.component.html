Data Analysis

For for Insurance Premium Predictor, we used R to run a multi-linear regression using variables that we determined had statistically significant predictive power for Prices. As you can see from the linear model summary below, we found that tobacco use , number of people insured and Optional Insurance premiums were by far the best predictors for Premiums.

The Pr(>|t|) indicates the p-value, or the probability that would observe such an extreme statistic such as the values seen in our dataset. All our variable's p-values are VERY small, which indicates good predictive power.

Also note the high Adjusted R-Squared value of 0.905 which indicates that our model adjusted for the noise from additional applications, can account for 90.5% of the variance which exists in our dataset.

The model above shows a function in R which helps us determine what the best possible multi-linear model is, by dropping one variable at a time and comparing their Adjusted R-Squared values. Looking at the topmost one, or the optimal model, we see that it includes the variables mentioned above.

Step 1: Unbiased Correlational Analysis

Given the diversity of variables, a good first approach is to analyze it for obvious correlations. A heatmap (Fig 1) was generated through the covariance of the pairings between every variable to each other. We omitted irrelevant or insignificant labels and converted non-numerical values into integers for plotting convenience.

The result showed two variables that are positively correlated with the quotes:

  • The number of people covered under one insurance plan tobacco usage
  • Tobacco usage, which is more strongly correlated than the prior

    • All other variables lacked a statistically significant correlation.

    Future Directions

    We could implement a better purchased plan predictive algorithm. Uncorrelated data implies independence; consequently, we can use more advanced machine learning techniques to analyze the independent variables involved in the dataset. Naive Bayes, using the multinomial method for discrete values or Bernoulli's method for binary variables or multinomial, is an option. Current methods allow for an estimated prediction of premium quotes for customer convenience. Data analytics have shown limited dependence on variables. However, this could be improved to be more accurate given more training data.
